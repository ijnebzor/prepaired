# PrepAIred — Roadmap

Each phase stands alone. Each ships with a security audit and its own pricing.

## Phase 1: Interview Simulator — DONE (v0.5.0)
 [x] 10-question interview (behavioural, governance, technical, scenario)
 [x] 5 interviewer personalities (Warm, Technical, Adversarial, HR/STAR, Curious)
 [x] Panel (2-3 interviewers) and 1:1 formats
 [x] 3-turn exchange per question (answer → follow-up → follow-up + score)
 [x] Timer per question, scales by type and difficulty (Relaxed / Realistic / Brutal)
 [x] 4-dimension scoring (Structure, Confidence, Technical Depth, Role Readiness) 1-10
 [x] Coaching tips, do-over system (max 3, -0.5 penalty each)
 [x] Multi-provider: Groq Llama 3.1 (free BYOK), Claude Haiku 4.5 (credits)
 [x] Credit system: Whop webhooks, email OTP auth via Resend
 [x] Credit packs: $2/1, $5/5, $10/15 sessions
 [x] JD ingestion (paste/PDF/URL), candidate profile input
 [x] Session export (JSON), outcome recording, landing page
 Stack: Single HTML, vanilla JS, Cloudflare Workers + KV
