PrepAIred — Worker Deploy Guide

Get paid.

Set up the Cloudflare Worker and Whop payment integration. This is all CLI — the only browser time is creating Whop products and wiring up the webhook.

⚡
Cloudflare Worker
Validates sessions, checks credit balance, proxies Anthropic API calls, decrements credits on success.
💳
Whop Payments
Handles checkout. Fires a webhook on purchase → Worker stores credits in KV keyed by email.
🔑
KV Credit Ledger
Cloudflare KV stores credits per email. 1 credit = 1 full interview session (10 questions × 3 turns).

Before you start
Prerequisites
Have these ready before running the script
Node.js 18+ — check with node --version
Cloudflare account — free tier at cloudflare.com
Whop seller account — whop.com/sell
Anthropic pool key — a separate key just for PrepAIred (not your personal one). Create at console.anthropic.com
Repo cloned locally — git clone https://github.com/ijnebzor/prepaired.git
Step 1
Run the setup script
This handles everything — KV creation, secrets, deploy, and patching index.html
Terminal 
# Clone and enter the worker directory
git clone https://github.com/ijnebzor/prepaired.git
cd prepaired/worker

# Make the script executable and run it
chmod +x setup.sh
./setup.sh
→
The script is interactive — it'll pause and ask for input at each step. It auto-patches index.html with your Worker URL and Whop URLs at the end.
⚠
On macOS, sed -i syntax differs. If you hit an error on the patch step, run: brew install gnu-sed then re-run the script, or do the index.html edits manually (Step 7 below).
Step 2 (alt)
Prefer manual steps?
Run these commands in order if you'd rather not use the script
Terminal

Install and login

cd prepaired/worker
npm install
npx wrangler login

Create KV namespace

npx wrangler kv:namespace create CREDITS
# → { binding = "CREDITS", id = "abc123abc123abc123" }

npx wrangler kv:namespace create CREDITS --preview
# → { binding = "CREDITS", preview_id = "def456def456def456" }

Copy both IDs into wrangler.toml — replace the two REPLACE_WITH_... placeholders.

Set secrets

# Your Anthropic pool key
npx wrangler secret put ANTHROPIC_API_KEY

# Generate and set session signing secret
node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
npx wrangler secret put SESSION_SECRET

# Whop webhook secret — set after creating the webhook in Step 5
npx wrangler secret put WHOP_WEBHOOK_SECRET

Deploy

npx wrangler deploy
# → Published prepaired-api (https://prepaired-api.YOURNAME.workers.dev)

# Verify it's live
curl https://prepaired-api.YOURNAME.workers.dev/health
# → {"status":"ok","version":"1.0.0"}

Step 4
Configure Whop webhook
Whop notifies your Worker every time someone buys credits
Browser + Terminal

In Whop: Dashboard → Developer → Webhooks → Add endpoint

Endpoint URL

https://prepaired-api.YOURNAME.workers.dev/webhook/whop

Events to subscribe

payment.succeeded
membership.created

After saving, Whop shows you a Signing Secret. Copy it, then:

npx wrangler secret put WHOP_WEBHOOK_SECRET
# Paste the signing secret when prompted
Step 5
Custom domain (optional but clean)
Point api.prepaired.ijneb.dev at your Worker
Browser + Terminal

In Cloudflare Dashboard → Workers & Pages → prepaired-api → Settings → Domains → Add Custom Domain:

api.prepaired.ijneb.dev

Cloudflare handles the DNS automatically since ijneb.dev is already on Cloudflare.

Then redeploy with the production env config:

npx wrangler deploy --env production
Step 6
Update index.html
Wire in your Worker URL and Whop checkout URLs, then push
Terminal 
# From the repo root
cd ..

# Replace Worker URL (use your actual URL)
sed -i "s|https://prepaired-api.ijnebzor.workers.dev|https://api.prepaired.ijneb.dev|g" index.html

# Replace Whop checkout URLs
sed -i "s|WHOP_SINGLE_URL|https://whop.com/checkout/plan_YOUR_SINGLE_ID|g" index.html
sed -i "s|WHOP_5PACK_URL|https://whop.com/checkout/plan_YOUR_5PACK_ID|g" index.html
sed -i "s|WHOP_15PACK_URL|https://whop.com/checkout/plan_YOUR_15PACK_ID|g" index.html

# Commit and push
git add index.html
git commit -m "config: wire up Worker URL and Whop checkout URLs"
git push
⚠
macOS: use sed -i '' instead of sed -i, or install gnu-sed with brew install gnu-sed && alias sed=gsed
Step 7
Test end to end
Grant yourself test credits and verify the full flow before going live
Terminal

Health check

curl https://api.prepaired.ijneb.dev/health
{"status":"ok","version":"1.0.0"}

Grant test credits to your email

npx wrangler kv:key put \
  --namespace-id=YOUR_KV_ID \
  "credits:you@youremail.com" \
  '{"credits":3,"email":"you@youremail.com","plan":"test","lastPurchase":"2026-03-20T00:00:00Z","totalPurchased":3}'

Verify the auth endpoint

curl -X POST https://api.prepaired.ijneb.dev/auth/verify \
  -H "Content-Type: application/json" \
  -d '{"email":"you@youremail.com"}'

{"ok":true,"token":"...","credits":3,"email":"you@youremail.com"}

Open the site and test the Credits tab

https://ijnebzor.github.io/prepaired

Click Get PrepAIred → switch to Use Credits tab → enter your email → verify → begin interview.
